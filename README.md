# CelebrityFaceRecognition

>**Project Name**- 
   CelebrityFaceRecognition

* **Aim**-
   Use this project to classify Movie Actor Personalities. 
 
* **Table of Contents**-
     Introduction,
     General info,
     Technologies,
     Algorithm/model,
     Important Libraries,
     Data Source.
        

* **Introduction**-
       In this Data Science project, We classify Movie Actor Personalities.
           
           1. Hrithik Roshan                                                 
           2. Jhonny Depp                                         
           3. Keanu Reeves                             
           4. Tom cruise                           
           5. Tom Hiddleston 

* **General info**-
  This  Project classify movie actor images using opencv by face structure.  

* **Technologies**-

       1. Python
       2. Numpy and OpenCV for data cleaning
       3. Matplotlib & Seaborn for data visualization
       4. Sklearn for model building
       5. Jupyter notebook as IDE


* **Algorithm/model**-
           SVM(Support vector machine)

* **Important Libraries**-
          Numpy,
          matplotlib,
          pandas,
          sklearn,
          Opencv,
          Seaborn.
          
 * **Data Source**-
                 Scrap images from google using imageye a chrome-extension
